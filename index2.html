Understanding Breaking point classes.

This is a sample text which would resize and structure itself(change of font color, font size and background color) based on the window resizeing / pixel resolutions.
In order to understand the responsive feature, open this html on browser and resize the window. Or use the developer tool and provide the various resolutions.

Below are the various resolution based classes

  • Small devices: sm => {{@media(min-width: 640px)}}
  • Medium devices: md => {{@media(min-width: 768px)}}
  • Large devices: lg => {{@media(min-width: 1024px)}}
  • Extra Large devices: xl => {{@media(min-width: 1280px)}}
  • Double Extra Large devices: 2xl => {{@media(min-width: 1536px)}}
